# 💳 Machine Learning Based Fraud Payment Detection System

A web-based Machine Learning application that detects fraudulent credit card transactions using the Random Forest algorithm. The application supports both single transaction prediction and bulk CSV file analysis through a Flask web interface.

---

## 📌 Features

- Detect fraudulent credit card transactions.
- Predict a single transaction using manual input.
- Upload a CSV file for bulk fraud detection.
- Display fraud, genuine, and total transaction counts.
- Show fraud risk level (Low, Medium, High).
- Interactive dashboard with charts.
- Professional and responsive user interface.

- Total Transactions: **284,807**
- Fraud Transactions: **492**
- Genuine Transactions: **284,315**

Dataset Source:
https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

## 🚀 How to Run the Project

### Clone Repository

```bash
git clone https://github.com/yourusername/Fraud-Payment-Detection.git
```

### Install Dependencies

```bash
pip install pandas numpy flask scikit-learn joblib
```

### Run the Project

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
